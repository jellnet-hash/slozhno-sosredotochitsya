# Сложно сосредоточиться

Учебный фронтенд-проект — адаптивная веб-страница с поддержкой нескольких тем оформления.

Проект посвящён проблеме концентрации внимания и демонстрирует работу с CSS-переменными, адаптивной вёрсткой и переключением тем без изменения JavaScript-кода.

## Функциональность
- Переключение тем: день / неон / авто
- Адаптивная вёрстка для мобильных, планшетов и десктопов
- Сохранение выбранной темы
- Семантическая HTML-разметка и состояния доступности

## Технологии
- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (готовый скрипт переключения тем)

## Как запустить проект
1. Клонировать репозиторий
2. Открыть файл `index.html` в браузере
